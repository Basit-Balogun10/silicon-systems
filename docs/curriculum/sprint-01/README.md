# Sprint 01 Living Curriculum Index

This folder tracks classroom progress from project start to completion using adaptive paths.

## How To Use

1. Follow the Core Path stages in order.
2. Apply Adaptive Branch Rules based on student answers.
3. Update stage status after each teaching block.
4. Record divergence paths, skips, and evidence artifacts.

- [Project B - Traffic Light Controller FSM](project-b-curriculum.md)

## Stage Status Legend

- `planned`: not started.
- `active`: currently teaching.
- `done`: completed with evidence.
- `diverged`: branched to remediation/deep-dive.
- `skipped`: intentionally skipped due demonstrated competency.
